SFML Map Gallery

Example imagery from various multibeam survey projects produced through the Sea Floor Mapping Lab. Full extent and closeup views of shaded relief grayscale and depth-colored multibeam bathymetry are included in a layout format. Click on thumbnail for a larger view. Many of these data sets are displayed in this web sites IMS viewer.

Note, the imagery displayed here is not to be used for navigation. See Data Download page for data access, or contact Pat Iampietro if you are interested in obtaining the original images or associated data not currently available on-line. Data includes FGDC-compliant metadata.
