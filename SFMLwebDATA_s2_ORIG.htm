SFML Data Library
Public access to SFML data is supported by CICORE funding
(Center for Integrative Coastal Observation, Research, and Education)

Data presented here include a series of remotely sensed images (multibeam, side scan sonar), derived data (bathymetric contours, grid analyses, etc.), habitat analyses, associated data sets (survey footprints, coastline), and FGDC metadata, grouped by survey location. The list of SURVEY LOCATIONS links to a separate download table for each region. The DATA LOCATOR MAP site icons are linked to the same Data (Download) Library pages and can be used for general searches. To download zipped data files from the tables, click on the Data Type name under the desired survey location. Data are presented in ESRI ArcGIS format (shapefiles, grids). Multibeam and sidescan images are Geotiffs. Map tips on the data type icon will display the name of the zipped file. These data are NOT to be used for navigational purposes.

Data Use: These data are not copyrighted; however in the event that you publish or display results using any of the mapping data available on this download site, we request that you include the following acknowledgment: Data used in this study were acquired, processed, archived, and distributed by the Seafloor Mapping Lab of California State University Monterey Bay.

: If you download SFML data from this website, please drop us a line, to let us know how you are using or plan to use these data. Thank you.
