SIVA Spatial Analysis & Image Processing Tools

SIVA geospatial software resources support remotely-sensed data acquisition and handling; image processing; collection, management, editing, analysis, and presentation of GIS data; surveying; and visualization of data through animated, three-dimensional display techniques including flythroughs supported by modeling scenarios.


GIS Analysis, Visualization, & Modeling

  • ESRI ArcGIS
  • TerraSeer Boundary Seer
  • Tarsier Environmental Modeling Software Framework (custom scripting - EcoViz)

Sonar Acquisition and Processing

  • CARIS HIPS & SIPS
  • SEA Swathplus
  • Hypack Navigation
  • Riegl Marine Laser Scanner
  • Trimble GPS and Mapping

Remote Sensing Image Analysis

  • Erdas IMAGINE
  • IVS 3D Fledermaus

Statistical Analysis Packages

  • SPSS
  • FragStats
  • R-Stats Computing
