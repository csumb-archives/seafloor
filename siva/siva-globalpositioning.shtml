Global Positioning System Data - TRS Base Station

CSUMB provides TRS GPS base station receiver data for 2 reference stations - located at the California State University Monterey Bay (CSUMB) and Elkhorn Slough Estuarine Research Reserve (ESNERR- shut down).

Follow the web links below to access GPS files available for postprocessing.
