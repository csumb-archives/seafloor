CSMP -- California Seafloor Mapping Program web site

CSMP is a collaborative, multi-institutional campaign creating the first comprehensive, high-resolution basemap of California's state waters (shoreline out to 3 nautical miles)

BlueMesa Dreamweaver template - Detail page template

This template is for detail pages and does not include the feature graphic. To use, simply create your pages from the template, add your content and update your navigation links.

Image Floated Left - This is the H2 tag

An image with the assigned class "fltlft" will be aligned with the left edge of the content div, with a 10 pixel right margin and a 10 pixel bottom margin.

Editable Metatags and Head Section

The template allows for editing of the Title, Description and Keywords tags for each page plus another editable area in the Head for inserting scripts and other Head content.

Editable Template Areas Included

The template has left sidebar and content editable areas built-in already, so you can be up and running very quickly. You can change any of the editable regions in the template to meet the needs of your website project.

Fireworks and Photoshop Graphics Files Included

We've included the original Fireworks file (also exported as a PSD) in which we created the the site images. It includes all the graphic elements in this template with all the slices already created. It will be very easy to customize this template to suit your needs. All the image elements are vector graphics, so you can change colors, gradients, shadows, etc very quickly to generate a new look.
