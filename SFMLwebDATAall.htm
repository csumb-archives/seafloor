SFML Data Download

Data presented here include a series of remotely sensed images (multibeam, side scan sonar), derived data (bathymetric contours, grid analyses, etc.), habitat analyses, associated data sets (survey footprints, coastline), and FGDC metadata, grouped by survey location. The regional list of survey locations links to the download table below. To download zipped data files from the table, click on the Data Type name under the desired survey location. Data are presented in ESRI ArcGIS format (shapefiles, grids). Multibeam and sidescan images are Geotifs. Map tips on the data type icon will display the name of the zipped file. These data are NOT to be used for navigational purposes.
Data Use: These data are not copyrighted; however in the event that you publish or display results using any of the mapping data available on this download site, we request that you include the following acknowledgment: Data used in this study were acquired, processed, archived, and distributed by the Sea Floor Mapping Lab of California State University Monterey Bay.
